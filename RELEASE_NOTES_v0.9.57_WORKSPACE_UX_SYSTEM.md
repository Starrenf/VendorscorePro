# VendorScorePro v0.9.57 - Workspace UX System

Deze build trekt het nieuwe UX-principe breder door in VendorScorePro.

## Ontwerpprincipe

VendorScorePro gebruikt vanaf deze build consistenter het principe:

**Overzicht eerst → Details gegroepeerd → Bewerken bewust → Brondata gescheiden van verrijking**

## Aangepast

- Navigatie aangescherpt: Applicaties als zelfstandig register in de hoofdnavigatie.
- Architectuur verplaatst naar analyse/meer-menu.
- Globaal UX-systeem toegevoegd voor registers, master-detail schermen, KPI-kaarten, leesrijen, bewerkvelden en statuspanelen.
- Softwarelandschap/applicatieregister gebruikt compactere master-detail layout.
- Bewerkvelden zijn visueel gegroepeerd in rustige invoerkaarten.
- Licentievelden blijven contextueel: uitgebreide licentie-informatie verschijnt alleen wanneer licentieplichtig op Ja staat.
- Presentatiemodus blijft ondersteund.
